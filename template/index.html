Welcome to Add Search Firefox!

How To

  1. Edit `searches.toml`
  2. Execute `python3 main.py`
  3. Open displayed URL (We are here!)
  4. Click URL bar or search box
  5. Type some text
  6. Click a search engine icon with green plus tips
  7. Done!
  8. Optionally, add search keywords in Firefox's preferences
